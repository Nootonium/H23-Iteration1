# Health

## Description

Afin de pouvoir rapidement tester le fonctionnement de l'application, l'équipe de Floppa vous demande d'ajouter un *endpoint* d'état (*health*) du système.

## Détails techniques

### Requête

**Route**

`GET /health`

### Réponse

**Status**

`200 OK`
